# 自动化青协网站

## 项目简介

这是一个专为自动化学院青年志愿者协会打造的温暖、现代化的官方网站。网站以志愿者精神为核心，采用温暖的色彩搭配和现代化的设计风格，突出展示青协的大型志愿活动和温暖形象。

## 🌟 特色功能

- **温暖的视觉设计** - 橙色、黄色、青绿色等温暖色调
- **响应式布局** - 完美适配桌面端和移动端
- **志愿活动展示** - 重点突出大型志愿活动
- **大事记时间轴** - 展示青协发展历程
- **在线报名系统** - 便捷的志愿者报名功能
- **活动相册** - 记录每一个温暖的瞬间

## 📱 页面结构

- **首页** - 青协风采展示和最新动态
- **志愿活动** - 活动展示、搜索和筛选
- **关于我们** - 青协介绍、组织架构、团队介绍
- **大事记** - 时间轴形式展示重要事件
- **活动相册** - 志愿活动照片展示
- **加入我们** - 在线报名和联系方式

## 🛠 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **路由管理**: React Router
- **图标库**: Lucide React
- **动画效果**: Framer Motion

## 🚀 快速开始

### 环境要求

- Node.js 16+
- npm 或 yarn

### 安装步骤

```bash
# 克隆项目
cd youth-volunteer-website

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎯 核心组件

- **Navbar** - 响应式导航栏
- **Footer** - 网站底部
- **志愿者卡片** - 活动展示卡片
- **温暖按钮** - 渐变按钮组件
- **时间轴** - 大事记展示

## 📱 响应式设计

网站采用移动优先的设计理念，确保在各种设备上都有良好的用户体验：

- 手机端 (< 768px)
- 平板端 (768px - 1024px)
- 桌面端 (> 1024px)

## 🚀 部署说明

1. 构建项目：`npm run build`
2. 上传 `dist` 目录到服务器
3. 配置 Web 服务器指向 `dist/index.html`
